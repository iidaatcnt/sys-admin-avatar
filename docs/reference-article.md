## Difyのステップまとめ

### 準備
1. https://dify.ai でアカウント作成（Googleアカウント可、無料200クレジット付き）
2. 設定 → モデルプロバイダー → OpenAI をインストール
3. ナレッジベース用のテキストファイルを用意

### Step 1: チャットボット作成
1. スタジオ → アプリを作成 → 最初から作成
2. アプリタイプ：「チャットフロー」を選択
3. LLMブロックをクリック → モデル：OpenAI / gpt-4o-mini
4. システムプロンプトを入力
5. プレビューでテスト

### Step 2: 知識を教える
1. ナレッジ → ナレッジベースを作成
2. テキストファイルをアップロード → 次へ → 保存して処理
3. チャットフローに戻り「開始」と「LLM」の間に「知識検索」ブロックを挿入
4. 知識検索ブロック：ナレッジベースを選択
5. LLMブロック → コンテキスト → 「知識検索 / result」を追加
6. プロンプトにコンテキスト変数を挿入（/ → コンテキスト）

### Step 3: 安全装置
1. 「開始」と「知識検索」の間に「IF/ELSE」ブロックを挿入
2. 条件：sys.query が NGワードを「含む」
3. IF側（NGワードあり）に「LLM（お断り用）」ブロックを追加
4. 「変数集約器」ブロックを追加し、両LLMから線をつなぐ
5. 回答ブロックで「変数集約器 / output」を選択

### Step 4: 公開
1. 右上「公開する」をクリック
2. アプリ設定で名前・説明・アイコンをカスタマイズ
3. URLをシェア！

### フローの流れ
```
NGワードあり → お断りLLM → 変数集約器 → 回答
NGワードなし → 知識検索 → 通常LLM  → 変数集約器 → 回答
```

### トラブルシューティング
- 回答が表示されない → 変数集約器の接続を確認、回答ブロックの変数を確認
- 知識検索が機能しない → ナレッジベースの処理が「完了」になっているか確認
