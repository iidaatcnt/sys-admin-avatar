【Claude Code とは】
Claude Code は、Anthropic が提供するターミナル（コマンドライン）で動くAIコーディングアシスタントです。
ファイルの編集、コード生成、バグ修正、コマンド実行など、開発作業をAIが丸ごとサポートしてくれます。
Antigravity（エクステンション版）とは別のツールで、ターミナル上で動作します。

【インストールの前提条件】
- Anthropicの有料プラン（Claude Pro以上）が必要です（無料プランでは利用できません）
- macOS 13.0（Ventura）以降、または Windows 10（1809）以降が必要です
- インターネット接続が必要です
- Windowsの場合は Git for Windows のインストールを推奨します

【Macでのインストール手順】
1. ターミナルを開く（Spotlight検索で「ターミナル」と入力）
2. 以下のコマンドをコピー＆ペーストしてEnterキーを押す：
   curl -fsSL https://claude.ai/install.sh | bash
3. インストールが完了するまで待つ（数十秒〜数分）
4. 「claude」とタイプしてEnterを押す
5. ブラウザが自動的に開くので、Anthropicにログインして認証を完了する
6. ターミナルに戻ると対話可能な状態になる

【Windowsでのインストール手順（PowerShell）】
1. スタートメニューで「PowerShell」を検索し、「管理者として実行」をクリック
2. 以下のコマンドをコピー＆ペーストしてEnterキーを押す：
   irm https://claude.ai/install.ps1 | iex
3. インストールが完了するまで待つ
4. 「claude」とタイプしてEnterを押す
5. ブラウザが開くのでAnthropicにログインして認証を完了する

【Windowsでのインストール手順（コマンドプロンプト）】
1. スタートメニューで「cmd」を検索し、「管理者として実行」をクリック
2. 以下のコマンドをコピー＆ペーストしてEnterキーを押す：
   curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
3. インストールが完了するまで待つ
4. 「claude」とタイプしてEnterを押す

【インストール確認コマンド】
- バージョン確認：claude --version
- 診断ツール（問題がある場合）：claude doctor

【よくある質問 - インストール編】
Q: npmでインストールする方法は使えますか？
A: npm を使った旧方法は現在「非推奨（レガシー）」となっています。これから始める方は curl や PowerShell のネイティブインストーラーを使ってください。

Q: インストール後に「claude: command not found」と表示される
A: ターミナルを一度閉じて再度開いてみてください。それでも解決しない場合は「claude doctor」コマンドで診断できます。

Q: 無料プランでも使えますか？
A: いいえ、Claude Pro以上の有料プランが必要です。月額20ドル（約3,000円）のProプランから利用可能です。

Q: VSCodeと一緒に使えますか？
A: はい！Claude Code はターミナルで動作しますが、VSCode の内蔵ターミナルからも利用できます。ただし、VSCode の拡張機能とは別物です。

Q: Windowsで「セキュリティエラー」が出る
A: PowerShellの実行ポリシーが制限されている可能性があります。PowerShellを管理者として実行し直してください。

Q: インストール中にエラーが出た
A: まず「claude doctor」を試してください。それでも解決しない場合は公式ドキュメント（https://docs.anthropic.com/ja/docs/claude-code）を参照してください。

Q: Macでターミナルはどこにある？
A: Spotlight検索（Command + スペース）で「ターミナル」と入力するか、アプリケーション → ユーティリティ → ターミナル から起動できます。

Q: インストールしたら何ができるの？
A: ターミナル上でAIと対話しながらコードを書いたり、ファイルを編集したり、コマンドを実行させたりできます。「このバグを直して」「このフォルダにある全ファイルの説明をして」など自然な日本語で指示できます。

Q: アップデートはどうすればいい？
A: ネイティブインストーラーを使った場合、バックグラウンドで自動更新されます。手動で最新版にしたい場合は、インストール時と同じコマンドをもう一度実行してください。

【初回起動の流れ】
1. ターミナルで「claude」と入力してEnter
2. 利用規約が表示されるので確認してEnter
3. ブラウザが開いて認証画面が表示される
4. Anthropicアカウントでログイン
5. 「Claude Codeへのアクセスを許可しますか？」→「許可」をクリック
6. ターミナルに戻ると「>」プロンプトが表示されて使用開始！

【便利な使い方の例】
- 「このコードのバグを見つけて修正して」
- 「このフォルダにあるPythonファイルをすべて説明して」

- 「package.jsonを読んでこのプロジェクトの概要を教えて」
- 「README.mdを日本語で書いて」
