# sys-admin-avatar 🤖

Dify を使って作るノーコードAIアバター（チャットボット）プロジェクト。

## 概要

プログラミングなしで、特定テーマに特化したAIチャットボットを作成・公開するためのリソース集です。

## 構成

```
sys-admin-avatar/
├── knowledge/          # Dify ナレッジベース用資料
│   └── claude-code-faq.txt   # Claude Code インストールFAQ
├── prompts/            # Difyシステムプロンプト集
└── README.md
```

## 使用ツール

- [Dify](https://dify.ai) - ノーコードAI開発プラットフォーム
